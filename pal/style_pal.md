# Style PAL

## 定義

Style PAL とは、描画様式や質感の変化を、
人物同一性・場面連続性・シーケンスの意味を壊さない範囲に留めるために、
許容される様式範囲・描画の癖・禁止様式 drift を固定する継続層である。

Style PAL は、単なる「画風指定」ではない。
見た目を派手にするための味付けでもない。
Style PAL は、様式を

- どの程度リアル寄りか
- どの程度アニメ寄りか
- 線、面、色、光をどう扱うか
- どこまで変えてよく、どこから別作品になるか

として固定し、人物同一性やシーケンスの意味を壊さずに
見た目を統一するための記述層である。

-----

## 目的

Style PAL の目的は、毎回魅力的な絵柄を作ることではない。
目的は、スタイルが変化しても以下の状態を維持することである。

- 同じ人物が同じ人物に見える
- 同じ場面が同じ場面に見える
- 同じ連作として読める
- 様式変更が人物 drift の口実にならない

Style PAL は、見た目を盛るためではなく、
見た目の変化を同一性の内部に留めるために存在する。

-----

## なぜ Style PAL が必要か

スタイル変更は、一見すると人物や背景とは独立した装飾に見える。
しかし実際には、スタイルはしばしば以下まで巻き込んで再解釈する：

- 顔立ち
- 骨格
- 年齢感
- 文化的類型
- 感情の温度

例：
- リアル寄りにすると欧米人化しやすい
- アニメ寄りにすると幼児化しやすい
- 油絵化すると generic beauty に流れやすい
- cinematic 化すると過剰にドラマチックになる
- フォトリアルへのジャンプで人物 drift が起こりやすい

したがって、スタイルも人物や場面を壊しうる層として
管理される必要がある。

-----

## Style PAL が固定するもの

### 1. Style Range

許容される様式の幅。

例：
- リアル寄りからアニメ寄りまでの範囲
- セミリアルの許容上限
- 線の強さ
- 面の整理の程度
- 質感の抽象化の程度

Style PAL は、一つの絶対的画風よりも
どこまでが同じ系列かを定義する方が重要である。

### 2. Rendering Signature

その作品系列らしさを作る描画の癖。

例：
- 柔らかい光
- 過度に硬くない輪郭
- 清潔な色面
- 控えめな陰影
- 肌を彫り込みすぎない
- 顔立ちを整えすぎない

Rendering Signature は、人物の骨格や気配と結びつく。

### 3. Style Priority

スタイルが他の層に従う順序。

Style PAL は基本的に以下の下位に置かれる：

1. Character PAL
2. Sequence PAL
3. Costume PAL
4. Background PAL

つまり、スタイルは重要だが
人物や意味を壊してまで守るものではない。

### 4. Forbidden Stylization

許容されない様式的 drift。

例：
- 欧米寄り formal beauty 化
- 幼児化
- generic anime heroine 化
- 過度な cinematic 化
- 油絵化による別人化
- glamour 強化
- 広告的な明るすぎる演出
- 「上手いが別人」になる様式変換
- フォトリアルへのジャンプ

-----

## 基本原則

### 1. スタイルは人物の上に乗るのであって、人物を置換しない

スタイル変更は、人物の芯を保ったまま上に重なるべきである。
スタイルを変えるたびに人物が再設計されてはならない。

### 2. スタイルは最後に効かせる

人物・背景・衣装・シーケンスの土台が固まる前に
スタイルを強くかけると drift しやすい。
スタイルは基本的に後段で調整する方が安全である。

### 3. リアル寄りとアニメ寄りの移動は drift を伴いやすい

途中まではリアル寄り、最後にアニメ寄りという運用は
有効なことがあるが、同時に style drift の危険も高い。
Style PAL は、この移動範囲と禁止線を明示する必要がある。

### 4. スタイル統一が sequence を殺さないようにする

スタイルが強すぎると、各コマの差分より絵柄の統一感が勝ち、
sequence が弱くなることがある。
Style PAL は、統一を与えるが、変化の意味を消してはならない。

### 5. フォトリアル jump を最重要 drift として扱う

Style PAL において最も起こりやすく、
かつ人物 drift を伴いやすいのは、フォトリアルへのジャンプである。
フォトリアル化に伴い、鼻筋強化・顎の細化・眼窩深化・
欧米人化・generic beauty 化が起こりやすい。
したがって、写実方向への移動は小さな段階で行い、
少しでも人物 drift が出た時点で停止する。

-----

## 他 PAL との関係

優先順位：

1. Character PAL
2. Sequence PAL
3. Costume PAL
4. Background PAL
5. Style PAL

スタイル変更で別人化した場合、
スタイルの成功ではなく人物同一性の失敗とみなす。

-----

## 起こしやすい Drift

- **Realism Drift**：リアル寄りにした結果、
  鼻筋強化・眼窩深化・顎の鋭化・欧米人化が起こること
- **Anime Drift**：アニメ寄りにした結果、
  目の拡大・顔の幼化・kawaii 化・記号的ヒロイン化が起こること
- **Painterly Drift**：油絵・厚塗り・芸術画風に寄せた結果、
  骨格が再解釈され generic beauty に流れること
- **Cinematic Drift**：映画的・広告的な演出に寄せた結果、
  感情が強すぎ generic romance visual になること
- **Photoreal Jump**：スタイルの連続性を越えて
  一気にフォトリアル側へ跳躍すること。
  人物 drift の最大入口として扱う。

-----

## 典型例

### 安定型
- セミリアル寄り
- 清潔で柔らかい光
- 顔の骨格を彫り込みすぎない
- アニメ寄りに寄せても幼くしすぎない
- 控えめで静かな表情を保つ

### 危険型
- リアル寄りで鼻筋や顎を強調
- 花嫁衣装と組み合わせて glamour 強化
- 油絵化で人物の芯を失う
- bright smile / radiant smile 系で表情 drift
- フォトリアル化で人物の文化的類型が変わる

-----

## 意義

Style PAL の意義は、様式を作品の表面ではなく
同一性を支える条件の一つとして扱えるようにすることにある。

これにより以下を成立させやすくなる：

- 別スタイルでも同じ人
- 別質感でも同じ場面
- 別描画でも同じ連作

Style PAL は、スタイル変更を自由度ではなく
制御可能な可変層へ変えるための層である。
