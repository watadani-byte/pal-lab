# Age Layer

## 定義

Age Layer とは、同じ人物を年齢変化の中でも同じ人物として保つために、
年齢ごとの許容変化・維持すべき芯・禁止 drift を整理した
Character PAL 内部の安定点層である。

Age Layer は、別人を作るためのものではない。
単なる年齢指定でもない。
Age Layer は、同じ人物が年齢変化の中でも同じ人物として読めるように、

- 何が自然に変わってよいか
- 何が変わってはならないか
- どこで類型 drift が起きやすいか

を整理し、再呼び出し時の安定点として扱うための層である。

-----

## 目的

Age Layer の目的は、年齢を正確に描き分けることではない。
目的は、年齢が変わっても以下の状態を維持することである。

- 同じ人物として読める
- 類型ジャンプしない
- 大人化の名目で別人化しない
- 幼児化や若返り drift を避ける
- 年齢変化が人格変質にならない

Age Layer は、年齢指定ではなく、
年齢をまたぐ同一性の運用のために存在する。

-----

## なぜ Age Layer が必要か

同じ人物を一つのアンカーだけで全年齢帯に渡って呼び出そうとすると、
モデルはしばしば以下を勝手に混ぜて再構成する：

- 若さ
- 大人っぽさ
- 花嫁らしさ
- フォーマルさ
- generic beauty

その結果、欧米人化・理想化・幼児化・年齢ジャンプ・類型ジャンプが
起こりやすくなる。

Age Layer はこの暴走を減らすために、
同じ人物の内部にある複数の安定点を明示的に持つ。

-----

## Age Layer が固定するもの

各 Age Layer は少なくとも次の3つを持つ。

### 1. 許容される変化

年齢によって自然に変わってよい要素。例：
- 頬の量感
- 表情の落ち着き
- 佇まいの余裕
- 首や肩まわりの静けさ
- 目元の圧のわずかな変化

### 2. 維持すべき芯

年齢が変わっても変えてはならない要素。例：
- 顔の重心
- 目元の温度
- 日本人女性としての自然な骨格
- 親しみやすい知性
- 冷たくなりすぎない落ち着き
- Character PAL の Identity Core

### 3. 年齢特有の禁止 drift

その年齢帯で起こりやすい誤った再解釈。例：
- 10代での幼児化
- 20代前半での kawaii 過剰化
- 20代半ばでの glamour drift
- 30歳前後での欧米寄り formal beauty drift
- 成熟の名目での人格変質

-----

## 基本構成

Age Layer は独立した別 PAL ではなく、
Character PAL の内部層として扱うのが原則である。

理由は、年齢を別 PAL として切り出すと、
モデルが年齢ごとに別人として再構成しやすくなるからである。

構造：

    Character PAL
      └── teens
      └── early 20s
      └── mid 20s
      └── around 30

-----

## 典型的な Age Layers

### teens

許容される変化：
- 頬の柔らかさ
- 軽い表情の動き
- 緊張感の残る若さ
- 全体の軽さ

禁止 drift：
- 幼児化
- 過剰な kawaii 化
- 目を大きくしすぎること
- 子供っぽい比例への崩れ

### early 20s

許容される変化：
- 少し整った輪郭
- 自意識の入り始めた佇まい
- 若さと落ち着きの中間

禁止 drift：
- glamorous beauty への吸着
- 大人化しすぎること
- generic heroine 化

### mid 20s

許容される変化：
- バランスの取れた落ち着き
- 静かな自信
- 感情表現の整理

禁止 drift：
- generic stylish woman 化
- 冷たい有能 archetype への drift
- 欧米寄りの美人類型への整理

### around 30

許容される変化：
- 表情の静かな余裕
- 少し成熟した空気
- 落ち着いた存在感
- 動作の柔らかい節度

禁止 drift：
- 鼻筋強化
- 眼窩深化
- 顎や頬骨の鋭化
- office-worker archetype 化
- generic glamorous adult woman 化
- 成熟の名目での別人化

-----

## 運用原則

### 1. 年齢は「別人」ではなく「別安定点」

Age Layer は、同じ人物の連続性を前提にする。
年齢が変わるたびに別の人物を設計しない。

### 2. 顔そのものより周辺で年齢感を支える

年齢感は、顔だけで無理に作ると drift しやすい。
以下の周辺層で年齢感を支える方が安全な場合がある：

- 服装
- 姿勢
- 表情の温度
- ライティング
- シチュエーション

### 3. 年齢上昇時は drift に特に注意する

年齢が上がるほど、欧米人化・理想化・類型ジャンプ・人格変質が
起こりやすい。
成熟方向の遷移では特に anti-drift を強く意識する。

### 4. 年齢変更も遷移として扱う

年齢変更は一気に飛ばさず、必要に応じて段階的に固定する。
年齢もまた、差分遷移の一種である。

-----

## 意義

Age Layer の意義は、人物の同一性を時間の中に延長できることにある。

これにより、10代・20代前半・20代半ば・30歳前後の同一人物を、
別人の集合ではなく同じ人の別時点として扱いやすくなる。

Age Layer は、人物同一性を静止した一点ではなく、
時間を含む連続体として運用するための層である。
