# Character Base Appearance

## Definition

Character Base Appearance is the foundational layer
that fixes the minimum visual constants required for
a person to remain readable as the same individual —
across changes in costume, background, composition,
and sequence.

This layer does not define personality, emotional state,
scene meaning, or rendering style.
It does not manage surface decoration or styling choices.

Character Base Appearance exists to preserve the minimum
visual identity of a person — the conditions that must
not be lost when surrounding elements change.

-----

## Purpose

The purpose of Character Base Appearance is not to
specify appearance like a blueprint.

It is to maintain stable person-reading under variable
conditions:

- The same person continues to be read as the same person
- Costume changes do not trigger body reinterpretation
- Background changes do not recode cultural reading
- Local visual features do not invoke nearby archetypes

This layer is not for optimizing beauty.
It is for stabilizing identity.

-----

## What This Layer Fixes

### 1. Cultural Reading Base

Character Base Appearance first fixes the cultural
reading frame within which the person should be understood.

Examples:

- Read as a Japanese woman
- Not reinterpreted as Western, even with lighter
  hair or eyes
- Visual features are not connected to westernized
  beauty conventions

The Cultural Reading Base is the top-level constraint
in this layer.

What is fixed here is not only which features are present,
but which reading frame those features are interpreted within.

### 2. Hair Anchor

Fixes hair-level constants that strongly contribute
to person identification: hair shape, color, and
facial framing configuration.

Examples:

- Warm brown, softly waved semi-long hair
- High ponytail
- Delicate side locks
- Layered fringe

The Hair Anchor is not a decorative variable that
follows fashion or costume.
It is a fixed condition supporting visual continuity.

### 3. Eye and Face Anchor

Fixes minimum constants for eyes, glasses, and
facial impression.

Examples:

- Soft, warmly lit brown eyes
- Thin-framed glasses
- Soft facial structure
- Eyes not deepened toward sculpted or cosmetically
  emphasized reads

The eye and face area is a high-drift zone and
requires explicit stabilization.

### 4. Body Balance Anchor

Fixes relative body reading and proportional impression.

Examples:

- Head reads slightly small relative to full body
- Neck is slender but not exaggerated
- Limbs are lean and long
- Chest is understated
- Waist reads high and narrow
- Thighs carry slight softness and volume

This is not an absolute proportion specification.
It is a relative body reading anchor.

Its role is to prevent costume, composition, or pose
changes from recasting the person into a different
body archetype.

-----

## Core Principles

### 1. Fix by relative reading, not absolute dimension

Character Base Appearance must not be treated as
a blueprint layer.

It does not require numerical ratios, skeletal
dimensions, or strict proportion enforcement.
It governs how the person should be read relatively —
not how they should be measured.

The goal is convergence toward the same person
under changing conditions, not dimensional control.

### 2. Visual features do not override cultural reading

Some visual features are inherently drift-prone.

Lighter brown hair, softly lit brown eyes, and
a moderately model-adjacent body balance can,
without constraint, invite reinterpretation toward
westernized visual templates.

Character Base Appearance prohibits this override.

Visual features are interpreted within the Cultural
Reading Base. They must not redefine it.

### 3. Costume does not redefine body

Costume is variable. Body is not.

Changes in dress, casual wear, workwear, or
formal attire must not trigger reinterpretation
of the person’s body reading.

Character Base Appearance functions as the fixed
body-side condition that precedes costume variation.

### 4. Composition changes are not license for drift

Full body, waist-up, bust-up — composition changes
alter visible information, not the person.

Camera distance, framing, and cropping must not
cause changes in facial reading, head-to-body ratio,
or body archetype.

-----

## Forbidden Reinterpretation

Character Base Appearance prohibits at minimum:

- Japanese woman → Western woman reread
- Lighter hair or eyes → westernized beauty template
- Long limbs → exaggerated high-fashion skeleton
- Slightly small head → non-realistic model proportion
- Narrow waist → hard, dry Western body code
- Softly lit eyes → hazel glamour or cosmetic emphasis
- Glasses → cold intellectual stereotype

These are not minor styling variations.
They are visual-level identity drift.

-----

## Drift Types This Layer Addresses

**Cultural Drift**
Local visual features trigger full recoding of the
person into a different cultural reading frame.

**Beauty-Code Drift**
Person-specific appearance is absorbed into a generic
glamorous template.

**Body Recasting**
Costume, pose, or composition causes the body itself
to be reread as a different person type.

**Cosmetic Drift**
Eyes, skin, or facial contour migrate toward an
over-defined, over-brightened beauty system.

-----

## Relation to Character PAL

Character Base Appearance and Character PAL are
closely related but carry distinct responsibilities.

Character Base Appearance fixes the minimum visual
constants of the person’s appearance.
Character PAL fixes how that person is read.

|Layer                    |Scope                                                                                           |
|-------------------------|------------------------------------------------------------------------------------------------|
|Character Base Appearance|Hair, eyes, glasses, facial framing, body balance, proportional impression                      |
|Character PAL            |Personality impression, behavioral reading, emotional register, interpretation drift suppression|

Character Base Appearance stabilizes how the person looks.
Character PAL stabilizes how the person is read.

-----

## Priority Order

Person reading is governed in the following order:

1. Cultural Reading Base
1. Visual anchors within that frame
1. Character PAL person reading
1. Costume, background, and sequence do not override
   the above

Lower layers must not override the identity conditions
established by higher layers.

In particular, visual features must not be used to
reverse-derive a different cultural identity.

-----

## Operational Significance

Character Base Appearance is required when:

- Costume changes destabilize body reading
- Background changes destabilize cultural reading
- Fixed features are inherently drift-prone
- The same person must be maintained across
  multiple visual contexts

This layer is a continuity layer, not a style layer.

-----

## One-Line Definition

Character Base Appearance is the foundational visual
layer that fixes hair, eyes, glasses, facial framing,
and relative body reading — with Cultural Reading Base
as the top constraint — so that the person remains
readable as the same individual across changes in
costume, background, composition, and sequence.
