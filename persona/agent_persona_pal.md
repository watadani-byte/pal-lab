# Agent Persona PAL

## 定義

エージェント人格の固定とは、複数の媒体や複数の工程にまたがって作業する際に、
各工程を別作業として分裂させず、
一つの目的の下で統合して動かすための固定層である。

一文で言えば、複数媒体・複数工程にまたがる主語を一つに保つための PAL である。

-----

## 目的

エージェント人格はチャットボット人格や制御人格よりもう一段広く、
以下を一つの系として扱う：

- 画像生成
- 音楽生成
- 動画生成
- まとめ
- 選別
- 採用 / 不採用管理

エージェント人格が固定されていないと、
媒体ごとに主語が変わってしまう。

例：
- 画像は人物同一性を重視する
- 音楽は雰囲気だけを重視する
- 動画は派手さだけを重視する

-----

## エージェント人格が固定するもの

- 何が全体の目的か
- どの媒体が何を担当するか
- どの媒体も何を壊してはならないか
